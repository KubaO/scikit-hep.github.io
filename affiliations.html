Affiliated packagesÂ¶

We take the good concept of an Affiliated package from the Astropy project. An affiliated package is a Python package that is not part of its core but is related to, and seen as part of, the Scikit-HEP project and community.

Affiliated packages are yet to be defined and the community is most welcome to get in touch.
